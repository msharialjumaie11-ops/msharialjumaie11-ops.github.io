This example shows how to use headings, paragraphs, and lists in HTML. The code is displayed using escaped characters so you can study it safely.

          <!DOCTYPE html>

üìò Introduction to HTML Tables

HTML tables are used to organize data into rows and columns. They‚Äôre great for displaying structured information like schedules, price lists, or tag references.

üß± Basic Table Structure

      <table>
        <thead>
          <tr>
            <th>Header 1</th>
            <th>Header 2</th>
          </tr>
        </thead>
        <tbody>
          <tr>
            <td>Data 1</td>
            <td>Data 2</td>
          </tr>
        </tbody>
      </table>

Explanation:

  • <table>: Defines the table.
  • <thead>: Contains header rows.
  • <tr>: Table row.
  • <th>: Table header cell (bold by default).
  • <td>: Table data cell.

üé® Formatting Tables with Attributes

      <table border="1" cellpadding="8" cellspacing="0">
        <thead>
          <tr>
            <th>Course</th>
            <th>Duration</th>
          </tr>
        </thead>
        <tbody>
          <tr>
            <td>HTML Basics</td>
            <td>2 Weeks</td>
          </tr>
          <tr>
            <td>JavaScript Intro</td>
            <td>3 Weeks</td>
          </tr>
        </tbody>
      </table>

Attributes:

  • border: Adds a border around cells.
  • cellpadding: Adds space inside cells.
  • cellspacing: Adds space between cells.
